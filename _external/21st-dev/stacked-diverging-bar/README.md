# StackedDivergingBar (Incident Report Card)

## What it is
Incident report card with a reaviz stacked diverging bar chart, 3 KPI metric rows with inline SVG icons and trend indicators, and dark mode support.

## What it's used for
- Incident/operations dashboards
- IT service management UIs
- Status reporting cards

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `"Incident Report"` | Card header title |
