# CryptoStatsCard (StatsCard2)

## What it is
Crypto dashboard card showing market cap, animated sparkline chart, Bitcoin dominance bar, and top coin price list with change indicators.

## What it's used for
- Crypto/finance dashboards
- Market overview widgets
- Portfolio trackers

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `marketCapUSD` | `number` | — | Total market cap in USD |
| `marketCapChange` | `number` | — | 24h change percentage |
| `chartData` | `number[]` | — | Sparkline data points |
| `dominanceData` | `Dominance[]` | — | Dominance breakdown |
| `coinData` | `Coin[]` | — | Top coins list |
| `currencySymbol` | `string` | `"USD"` | Currency label |
| `className` | `string` | — | Additional CSS classes |
