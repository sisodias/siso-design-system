# InteractiveCalendar

## What it is
An animated calendar grid component with hover interactions and a bookings list view, built with framer-motion.

## What it's used for
- Displaying a monthly calendar with meeting/booking indicators
- Interactive day hover with sorting animations
- Toggle between grid view and detailed bookings list
- Dark-themed scheduling interface

## Dependencies
- `framer-motion` - animations
- `lucide-react` - icons (Columns3, Grid)
