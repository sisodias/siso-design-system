# NotificationCard

## What it is
Animated notification card with staggered reveal: avatar, timeline connector, message bubble, and earnings summary — driven by framer-motion delays.

## What it's used for
- AI/earnings notifications
- Activity alerts
- Sequential message reveals

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `aiName` | `string` | `"AI name"` | Name displayed in header |
| `userName` | `string` | `"User"` | User name in message |
| `paperTopic` | `string` | `"general topic"` | Topic mentioned |
| `doctorName` | `string` | `"AI Doctor"` | Doctor name |
| `earnings` | `string` | `"$0.20c"` | Earnings amount |
| `weekTotal` | `string` | `"$400"` | Week total |
