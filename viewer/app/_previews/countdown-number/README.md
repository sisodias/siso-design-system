# AnimatedNumberCountdown

## What it is
A countdown timer that animates numbers smoothly using `@number-flow/react` and `framer-motion`.

## What it's used for
- Event countdowns (launches, sales, deadlines)
- Timers with animated digit transitions
- Real-time tracking displays

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `endDate` | `Date` | required | Target date to count down to |
| `startDate` | `Date` | `new Date()` | Start date for countdown |
| `className` | `string` | — | Additional CSS classes |
