# AnimatedCardChart

## What it is
A layered 3D glass-morphism card with animated bar charts that scale on hover, gradient overlays, and floating tooltip panels.

## What it's used for
- Dashboard stat cards
- Analytics visualizations
- Premium product UI with rich hover interactions

## Exports
- `AnimatedCard`, `CardBody`, `CardTitle`, `CardDescription`, `CardVisual` — card primitives
- `Visual3` — layered chart visual with configurable `mainColor`, `secondaryColor`, `gridColor`
