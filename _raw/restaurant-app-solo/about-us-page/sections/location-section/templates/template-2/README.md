# Location Template · Template 2

**Platform:** Mobile — all SISO-harvested components are mobile-first. Desktop layouts (if any) are secondary variants inside the same file. Use on mobile surfaces only unless explicitly redesigned.

- **Best for**: Hospitality brands wanting an editorial narrative with sticky map preview.
- **Layout**: Concierge column (address, contacts, hours) beside a tall map card that stays in view on desktop.
- **Content fields**: `pillText`, `title`, `subtitle`, `address`, `map`, `contacts`, `operatingHours`, `hoursSummary`, `parkingInfo`.
- **Notes**: First contacts render in the ordered list; provide at least one contact method to unlock the quick links block.
