# BarChart (shadcn/charts)

## What it is
Shadcn-style chart system wrapping recharts with `ChartContainer`, `ChartTooltip`, `ChartLegend`, CSS variable theming, and dark mode support via `ChartStyle`.

## What it's used for
- Analytics dashboards
- Data visualization with theming
- Bar, line, area charts

## Sub-components exported
`ChartContainer`, `ChartTooltip`, `ChartTooltipContent`, `ChartLegend`, `ChartLegendContent`, `ChartStyle`
