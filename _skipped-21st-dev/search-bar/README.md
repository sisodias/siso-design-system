# SearchBar

## What it is
An animated search bar with gooey particle effects, autocomplete suggestions, and rich micro-interactions using framer-motion.

## What it's used for
- Search interfaces with visual flair
- Autocomplete/dropdown suggestions
- Interactive form elements with animations

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | `"Search..."` | Input placeholder text |
| `onSearch` | `(query: string) => void` | — | Callback when search is submitted |
