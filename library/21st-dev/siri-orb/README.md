# SiriOrb

## What it is
Apple Siri-style animated orb with CSS `@property` conic gradients, blur/contrast filters, and interactive size/speed controls.

## What it's used for
- AI assistant avatars
- Loading/idle state indicators
- Ambient visual accents

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `string` | `"192px"` | Diameter of the orb |
| `className` | `string` | — | Additional CSS classes |
| `colors` | `{bg,c1,c2,c3}` | oklch defaults | Color stops for the orb |
| `animationDuration` | `number` | `20` | Full rotation duration in seconds |
