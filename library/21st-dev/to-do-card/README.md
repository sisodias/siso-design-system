# TodoCard

## What it is
A polished daily task list card with live date/time header, animated checkbox interactions, and confetti celebration when all tasks are complete.

## What it's used for
- Personal task management widgets
- Daily planners
- Productivity dashboards

## Props
None — the component is fully self-contained with internal state.

## Features
- Live clock with date/time header
- Gradient header changes from yellow (incomplete) to emerald (all done)
- Animated checkbox with custom checkmark
- Confetti animation on completion (respects `prefers-reduced-motion`)
- Reset button to restore initial state
- Radial dot background pattern
