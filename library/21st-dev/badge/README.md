# Badge

## What it is
A shadcn/ui badge component using `class-variance-authority` for variant-based styling.

## What it's used for
- Labels and tags
- Status indicators
- Category markers

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `default \| secondary \| destructive \| outline` | `default` | Visual style variant |
| `className` | `string` | — | Additional CSS classes |
