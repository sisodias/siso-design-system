# AnomalyHeatmap

## What it is
A data visualization card displaying a grid of cells with intensity-based coloring and hover-reactive cell scaling.

## What it's used for
- Security monitoring dashboards
- Anomaly detection UIs
- Signal intensity visualization
- Time-window analysis displays

## Props
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `rows` | `number` | `6` | Number of rows in the heatmap grid |
| `cols` | `number` | `10` | Number of columns in the heatmap grid |
| `cardTitle` | `string` | `"Anomaly heatmap"` | Card heading |
| `cardDescription` | `string` | `"Intensity of suspicious signals across time windows."` | Card subheading |
